# Homebrew Tap

Homebrew formulae and casks for my apps.

## Install

```bash
brew tap volodymyrsmirnov/tap
```

## Apps

| Name | Type | Description |
|------|------|-------------|
| [diriger](https://github.com/volodymyrsmirnov/diriger) | Cask | Menu bar app for quickly switching between Google Chrome profiles |
| [dotfather](https://github.com/volodymyrsmirnov/dotfather) | Formula | Lightweight symlink-based dotfile manager |
| [mcp-bin](https://github.com/volodymyrsmirnov/mcp-bin) | Formula | Turn MCP server tools into CLI commands |

```bash
brew install diriger        # latest
brew install dotfather      # latest
brew install mcp-bin        # latest
brew install mcp-bin@0.2.1  # specific version
```
